# 📱 Как получить APK — инструкция за 5 шагов

## Шаг 1 — Создай репозиторий на GitHub

1. Зайди на https://github.com/new
2. Repository name: `BankPushForwarder`
3. Оставь **Public** (или Private — не важно)
4. Нажми **Create repository**

---

## Шаг 2 — Загрузи файлы проекта

На странице нового репозитория нажми **"uploading an existing file"**

Перетащи ВСЕ файлы и папки из этого архива:
```
.github/
app/
gradle/
build.gradle.kts
settings.gradle.kts
README.md
```

> ⚠️ Важно: папку `.github` тоже нужно загрузить — в ней лежит инструкция для сборки.
> Если проводник Windows скрывает папки с точкой — включи "Показывать скрытые элементы" в Проводнике.

Нажми **Commit changes**

---

## Шаг 3 — Дождись сборки (5-7 минут)

1. Перейди во вкладку **Actions** в репозитории
2. Увидишь запущенный процесс "Build APK" — жди зелёной галочки ✅
3. Если видишь красный крестик ❌ — напиши мне, разберёмся

---

## Шаг 4 — Скачай APK

1. Кликни на завершённый workflow "Build APK"
2. Внизу страницы в разделе **Artifacts** найди `BankPushForwarder-debug`
3. Нажми — скачается ZIP с APK внутри

---

## Шаг 5 — Установи на Android

1. Распакуй ZIP — внутри файл `app-debug.apk`
2. Перешли его на телефон (через Telegram, почту, USB)
3. Открой на телефоне — Android спросит разрешение на установку из неизвестных источников
4. Разреши и установи

---

## После установки

1. Открой приложение **Bank Push Forwarder**
2. Нажми **«Включить»** → в настройках Android найди приложение и включи доступ к уведомлениям
3. Введи Bot Token и Chat ID → **«Сохранить»** → **«🧪 Тест»**
4. Готово — банковские пуши полетят в Telegram!
